제로봇 명령어

    !도움말

    제로봇의 모든 명령어를 보실수 있습니다.


    !핑

    현제 제로봇의 핑을 볼수있습니다.


    !인증

    캡챠를 이용한 인증 채널을 생성하실 수 있습니다.


    !필터링 설정

    필터링 설정방법을 알려줍니다.


    !필터링 로그채널

    필터링 감지시 기록되는 채널을 설정합니다.


    !필터링 로그채널삭제

    필터링 감지시 기록되는 채널설정을 삭제합니다.


    !서버로그

    서버에서 일어난 모든 로그를 저장&보여줍니다.(Dyno봇 생각하시면됨)


    !레벨 생성

    레벨장 채널을 생성합니다..


    !레벨업 설정

    서버내에서 레벨업 활성화 비활성화를 설정합니다.


    !레벨업 채널설정

    레벨업시 알림이 뜨는채널을 설정할수 있습니다.


    !레벨업 채널삭제

    기존에 설정한 레벨업 채널설정을 삭제합니다.


    !방송알림 등록

    명령어를 입력한 채널에 설정한 스트리머의 트위치 방송이 켜질때마다 알려줍니다.


    !방송알림 삭제

    기존에 등록했던 방송알림 정보를 삭제합니다.


    !입장채널 설정

    명령어를 입력한 채널을 입장채널을 설정하여
    서버에 입장한유저를 채널에 표시합니다.


    !입장채널 삭제

    명령어를 입력한 채널에 더이상 서버에 입장한유저를 표시하지 않습니다.


    !퇴장채널 설정

    명령어를 입력한 채널을 퇴장채널을 설정하여
    서버에서 퇴장한유저를 채널에 표시합니다.


    !퇴장채널 삭제

    명령어를 입력한 채널에 더이상 서버에서 퇴장한유저를 표시하지 않습니다.


    !입장DM 설정

    서버에 입장한 유저에게 설정한 메세지를 DM으로 전송합니다.


    !공지

    특정채널에 원하는 내용으로 공지를 전송합니다.


    !삭제

    입력한 숫자만큼 메세지가 삭제됩니다.


    !슬로우모드

    입력한 시간(초) 만큼 채널에 슬로우모드를 적용합니다.


    !이모지 확대

    서버내에서 이모지 확대 옵션을 활성화/비활성화 시킬수 있습니다.
    기본이 비활성화 상태입니다.


    !뮤트

    특정 유저에게 명령어를 사용한 카테고리에 채팅금지를 부여합니다.


    !언뮤트

    특정 유저에게 명령어를 사용한 카테고리에 채팅금지를 해제합니다.


    !킥

    특정유저를 서버에서 킥 시킵니다.


    !밴

    특정유저를 서버에서 밴 시킵니다.


    !경고

    경고관련 명령어를 출력합니다.


    !경고 가입

    서버에서 경고 관련명령어를 사용할수있게 합니다.


    !경고 부여

    특정유저에게 경고를 1회 부여합니다.
    경고는 3회 누적시 서버에서 밴처리 됩니다.


    !경고 회수

    특정 유저의 누적경고를 1회 삭제합니다.


    !경고 조회

    특정 유저의 경고횟수를 조회합니다.


    !경고 목록

    서버내에서 경고받은 유저를 보여줍니다.


    !경고 로그채널

    이 명령어를 이용시 명령어를 사용한 채널에 경로부여, 경고회수 기록이 남게됩니다.


    !경고 로그채널삭제

    경고 로그채널을 삭제합니다.
